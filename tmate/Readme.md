# Prerequisite
- libssh 
``` sudo apt-get install -y libssh-dev```
- msgpack
``` sudo apt install libmsgpack-dev ```
``` sudo apt-get install libevent-dev libncurses-dev pkg-config ```
``` sudo apt-get install autotools-dev ```
``` sudo apt-get install automake ```
